# 导演核心方法论

## 一、叙事目的分析法

### 什么是叙事目的

叙事目的是一场戏（或一个镜头）在全片叙事中承担的功能。它回答的问题是：**"这段内容为什么必须存在？如果删掉它，全片会缺少什么？"**

叙事目的有两个层面，分镜表中应同时覆盖：

**结构层（叙事功能）：** 这场戏在全片结构中的位置和作用。
- 交代信息（人物、空间、时间、规则）
- 建立/强化角色特征
- 推进情节（制造冲突、升级冲突、翻转）
- 埋设/回扣伏笔
- 呈现主题论点
- 首尾呼应 / 结构对称
- 情绪转场 / 节奏调节（呼吸点）

**导演执行层（视听意图）：** 导演希望通过这场戏在观众身上制造什么效果。
- 制造不安感 / 压迫感 / 窒息感
- 控制观众视线 / 引导注意力
- 营造某种氛围（温暖、荒凉、疏离……）
- 制造时间感（压缩、拉伸、停滞）
- 用视觉隐喻传达不可言说的内容
- 用节奏变化标记叙事层次的切换
- 建立或打破观众的预期

### 从剧本提取叙事目的的方法

**第一遍：结构性通读**

把整个剧本读一遍，不停留在细节上，标注每场戏的核心事件和价值转变（McKee标准：这场戏开始时和结束时，核心价值是否不同？）。完成后得到一份"骨架图"——全片的事件脊柱。

**第二遍：功能标注**

逐场标注结构层叙事目的。每场戏至少有一个，最多不超过三个。如果一场戏找不到任何叙事目的——它可能应该被删除或合并。

常见标注格式：
```
【场景3：医院走廊 / 夜】
结构功能：揭示角色B的秘密（信息释放），同时完成A对B的信任→怀疑的价值转变
```

**第三遍：导演意图叠加**

在结构功能基础上，叠加导演执行层的意图——"我要用什么视听手段让这个结构功能生效？"

```
【场景3：医院走廊 / 夜】
结构功能：揭示角色B的秘密，A的信任→怀疑
导演意图：用荧光灯的闪烁制造不安，走廊的纵深构图暗示秘密的深度，A发现秘密时不用特写而是保持中景——让观众自己在A的肢体语言中读出震惊
```

**第四遍：合并为分镜表叙事目的列**

将两个层面压缩为一句完整的视听分析，写入分镜表的"叙事目的"列。

好的叙事目的写法示例（来自用户真实作品）：
- "交代故事发生的空间背景与核心视觉设定（蓝线及其初始高度）。"
- "展现麻木看客的面目。"
- "形成视觉与叙事上的双重闭环，开启属于Steve的新一轮情感循环。"
- "用一个毫无实际意义的模拟动作收尾，将系统的荒谬与虚无感推至极点。"
- "视觉奇观升华主旨。将个体行为扩展为群体现象，隐喻现代都市男女在标准化社会中，连寻找爱情和庇护所的方式都是批量复制和虚假的。"

写法要点：
- 用一句话同时覆盖"做什么"和"为什么这么做"
- 具体到视听手段（"用背对背构图展现疏离"而非"展现他们的关系变化"）
- 当存在隐喻/母题时，点明其叙事功能（"呼应片名的两层含义"）
- 语气客观分析，不煽情

### 叙事目的的自检

标注完所有场景的叙事目的后，做以下检查：

- [ ] 是否每场戏都有至少一个叙事目的？（无目的的戏应删除或合并）
- [ ] 全片的叙事目的是否均匀分布？（信息交代是否过度集中在前段？情感高潮是否有足够铺垫？）
- [ ] 伏笔/回扣是否成对出现？（埋了的伏笔必须有回扣点）
- [ ] 主题是否通过视听动作而非台词传达？（如果叙事目的中大量依赖"角色说出XX"，说明视听化不足）
- [ ] 呼吸点是否存在？（连续三场高强度叙事目的之后需要一场节奏调节）

## 二、导演定调方法论

### 六维定调法

导演定调不是只选一个"类型"——而是从六个维度综合考量，从每个维度中提取合适的导演方法论，最终融合为一套独属于本项目的视听方案。

六个维度对应六个类型库文件：

| 维度 | 核心问题 | 参考文件 |
|------|---------|---------|
| A. 情绪基调 | 影片的情感温度和审美姿态是什么？ | `genre-A-mood.md` |
| B. 类型片 | 遵循什么类型惯例？在哪里打破？ | `genre-B-genre.md` |
| C. 动作与对抗 | 身体对抗如何产生冲击力和叙事意义？ | `genre-C-action.md` |
| D. 题材与关系 | 如何用视听语言呈现人与人的关系和情感？ | `genre-D-theme.md` |
| E. 叙事形式 | 叙事形式本身对视听有什么特殊要求？ | `genre-E-form.md` |
| F. 社会视角 | 影片如何处理与现实社会/历史的关系？ | `genre-F-social.md` |

**不是每个维度都需要选择。** 一个项目通常涉及2-4个维度。例如：
- 一部黑色幽默公路片 → A7黑色幽默（科恩兄弟）+ E2公路片（佩恩）+ D1爱情（林克莱特的对话方式）
- 一部科幻悲剧 → B7科幻（维伦纽瓦）+ A5悲剧（李沧东的日常磨损）
- 一部武侠主旋律 → C2武侠（胡金铨）+ F4主旋律（中国精品方向）

从每个相关维度中借鉴1-2位导演的具体方法论，融合时确定一个"主导导演参考"和若干"局部借鉴"。

### 引导式提问策略

**不要直接把维度和导演列表给用户。** 通过引导式提问逐步锁定各维度方向：

**第一轮：情绪和类型的大方向**
> "你希望观众看完这部片子最大的感受是什么？"
> "如果只用一个词形容这部片子的'气质'，你会选什么？"

根据回答锁定A（情绪基调）和B（类型）方向，内部调用对应导演方法论。

**第二轮：视听风格的偏好**
> "有没有哪部电影的'画面感觉'是你想要的？不一定是同类型。"
> "你最不想要的是什么？太花哨的运镜？太慢的节奏？太暗的画面？"

根据参考片或排除项，精确定位到具体导演。如果用户提到的参考片对应的导演在库中有方法论，直接调用。

**第三轮：题材和形式的特殊需求**
> "这个故事有没有什么特殊的叙事形式？比如多时间线、伪纪录片、公路结构？"
> "核心关系是哪一种？家庭、爱情、对手？"

锁定D（题材）和E（形式）维度。

**第四轮：社会和历史维度（如需要）**
> "这个故事和现实的关系是什么？需要真实感还是可以风格化？"

锁定F维度。

**每一轮提问后，内部从对应的genre文件中提取具体导演方法论，向用户呈现2-3个有差异的导演方向。** 方向的呈现方式是"描述这种视听风格会让影片变成什么样"，而非"这是XX导演的风格"——导演名字可以作为参考点提及，但核心是可执行的视听策略。

### 定调输出格式

确认后，用以下格式输出导演定调：

```
═══════════════════════════════════
🎬 导演定调 | [项目名称]
═══════════════════════════════════

【视听姿态】摄影机作为[XX]，与角色保持[XX]距离

【维度参考】
  情绪基调：[XX]（核心参考：[导演]的[具体方法]）
  类型方向：[XX]（核心参考：[导演]的[具体方法]）
  题材关系：[XX]（局部借鉴：[导演]的[具体方法]，用于[XX段落]）
  [其他相关维度...]

【融合方案】
  整体基调：以[导演A]的[XX方法]为底色
  [段落/场景]：借鉴[导演B]的[XX方法]
  [段落/场景]：借鉴[导演C]的[XX方法]

【镜头语言】[固定/手持/混合] + [长镜头/快切/混合] 为主
【构图方向】[对称/不对称/自然] + [留白/密集]
【色调光线】[暖/冷/去饱和] + [自然光/戏剧光/实际光源]
【剪辑节奏】[快/慢/变化] + [线性/非线性]
【声音设计】配乐[驱动/辅助/极少]，环境声[写实/风格化]
【叙事视角】[全知/限制/多视角] + 观众信息量[大于/等于/小于]角色

【意象体系】
  视觉母题：[贯穿全片反复出现的视觉元素]
  意象方向：[与主题/角色弧光呼应的环境意象风格]
  需要突出强化的视觉元素：[哪些物件/空间/动作在全片中有特殊分量]

═══════════════════════════════════
```

## 三、节奏规划

### 双轨节奏系统（继承自编剧技能）

节奏分为外部情节节奏和内在情感节奏两条轨道。导演阶段在编剧已建立的节奏基础上做**执行层的细化**：

**编剧阶段已确定的：**
- 每场戏的情节密度（松/中/紧）
- 每场戏的情感分量（轻/中/重）
- 全片节奏波浪图的大致走向

**导演阶段要确定的：**
- 每场戏的**镜头密度**（长镜头为主 vs 快切为主 vs 混合）
- 每场戏的**景别重心**（远全景 vs 中景 vs 特写）
- 每场戏的**时长分配**（在全片总时长中占多少比例）
- 场景之间的**转场策略**（硬切 vs 叠化 vs 匹配剪辑 vs 音乐桥）
- 每场戏内的**镜头组划分**（哪些镜头构成一个叙事单元）

### 镜头组（Shot Group）

**一个叙事目的可能由一组镜头共同完成。** 镜头组是节奏规划和分镜拆解之间的桥梁——它比"场景"小，比"单镜头"大，是一组逻辑紧密相接的镜头序列。

#### 什么是镜头组

镜头组是服务于同一个叙事目的的连续镜头序列。组内镜头之间的剪辑逻辑是紧密的——它们共同构成一个完整的视听段落，拆散就失去意义。

示例（来自实际分镜）：
```
镜头组A "建立角色人设"（3镜，共6秒）
  镜1(2秒): 亚瑟手持奖状的复古肖像照 → 引出主角
  镜2(2秒): 亚瑟胸前工作牌特写"99.9% ACCURACY" → 具象化特征
  镜3(2秒): 亚瑟坐着手持文件，神情骄傲自信 → 展现性格底色

镜头组B "全球媒体蒙太奇"（5镜，共8秒）
  镜1-5: 俄RT/日NHK/美CNN/法France24/意Rai 各国主播依次播报
  → 整组服务于同一个叙事目的：展现事件的全球影响力

镜头组C "时间线倒流"（3镜，共4秒）
  镜1-3: 快切闪回片段，画面不连贯
  → 整组服务于：暗示美好关系出现裂痕，用时间的混乱感传达情感的混乱
```

#### 镜头组的类型

**蒙太奇组：** 多个并列的画面快切组合，展示"多个同类事物"。组内每镜1-3秒，镜与镜之间是并列关系。典型用途：建立氛围、展示规模、压缩时间。

**递进组：** 镜头之间是层层递进的关系，每一镜在上一镜基础上增加信息或强化情绪。典型用途：角色介绍、设定展示、情绪升级。

**因果组：** 镜头之间是动作→反应的因果关系，紧密衔接。典型用途：对话正反打、动作场景的攻防。

**对比组：** 两个或多个画面形成对比关系。典型用途：前后呼应、并行叙事、反讽。

#### 在节奏规划中使用镜头组

节奏规划阶段应标注每场戏内的镜头组划分——这决定了后续分镜拆解时哪些镜头是"一组"的：

```
【场景2：邮局 / 白天】
  情节节奏：中（角色介绍）
  情感节奏：轻（轻松的铺垫）
  ──────────────────────────
  镜头组划分：
    组A"人物名片"（递进组，3-4镜，~6秒）: 快切建立角色身份和核心特征
    组B"工作蒙太奇"（蒙太奇组，4-5镜，~8秒）: 展示角色的工作能力
    → 全场景预估：~14秒
```

### 场景级节奏标注

在导演阶段，每场戏应标注：

```
【场景5：天台 / 傍晚】
  情节节奏：松（一次安静的对话）
  情感节奏：重（角色第一次面对真相）
  ──────────────────────────
  镜头策略：固定机位为主，中景保持两人空间关系
  景别重心：中景 → 最后一句台词时切近景
  预估时长：2分钟
  转场策略：硬切到下一场的对比空间
  节奏意图：情节松+情感重的错位——画面越平静，情感越沉
```

### 全片节奏波浪图（导演版）

在编剧版节奏波浪图基础上，叠加导演执行层的信息：

```
示意（纵轴=强度，横轴=时间线）：

情节节奏:  ─╱╲─────╱╲──╱╲╱╲╲──────╱╲╱╲╱╲╲
情感节奏:  ──────╱╲───────╱╲──╱╲╱╲───────╲╱╲
镜头密度:  ─────╱╲─╱╲──╱╲╱╲╲╲─────╱╲╱╲╱╲╲╲
景别距离:  远──中──近──中──远──近近近──中──近近近

              ↑建置     ↑发展        ↑高潮   ↑结局
```

### 节奏诊断清单

- [ ] 高潮段的镜头密度是否与前后形成对比？（高潮应是全片镜头密度最高或最低的段落——两种都可以，但不能与前后段持平）
- [ ] 是否存在连续3分钟以上的单一景别段落？（如果有，需要有明确的节奏意图）
- [ ] 转场策略是否有全片一致性？（同类型的场景切换用同一种转场）
- [ ] 呼吸场景是否被保留？（不要让节奏优化把所有安静时刻都压缩掉）
- [ ] 最长的单场戏是否有足够的内部节奏变化？（一个3分钟的长场景内部也需要松紧变化）
- [ ] **克制不等于匀速**——即使是最文艺的长镜头风格，全片也必须有节奏变奏（见下方专项警告）

### ⚠ 节奏陷阱警告：克制≠匀速

**问题描述：** 当导演定调偏向文艺/日常写实/诗意方向时，容易把"克制"理解为"全程长镜头、全程慢节奏、全程同一种呼吸感"，导致全片变成一条没有起伏的直线。

**核心认知：** 是枝裕和的节奏不是匀速的。《小偷家族》有安静的吃饭戏，也有海边的快速剪辑，也有最后审讯室里突然收紧的正反打。王家卫有漫长的凝视，也有《重庆森林》里疯狂的手持快切。塔可夫斯基的五分钟长镜头之所以有力量，恰恰因为它和其他段落的节奏不同。

**克制的真正含义是"控制变化的幅度"，不是"没有变化"。**

具体检查：
- 全片是否有至少一处与整体基调明显不同的节奏段落？（安静片中的一段快切、长镜头风格中的一次跳切蒙太奇）——这个"异类"段落往往是全片最有力的时刻。
- 长镜头内部是否有"事件"？（角色进入/离开画面、焦点转换、光线变化、声音突变）——一个5分钟的固定镜头如果画面中什么都不变，那不是克制，是停滞。
- 对话场景是否全部用同一种剪辑节奏？——即使整体偏慢，不同的对话场景应该有不同的切镜频率，取决于对话的内在紧张度。

### 镜头设计逻辑

切镜连接逻辑、动作-反应结构、对白场景处理方式、分镜 Double Check 机制均独立存放于 `references/shot-design.md`，在分镜拆解阶段（第四步）读取。

## 四、剧本微调原则

### 视听化修正（前置步骤）

在进入正式微调之前，先做一轮纯粹的视听化修正——**只修改无法被摄影机拍到的内容**：

- "角色觉得XXX"、"角色感受到XXX"、"角色意识到XXX" → 改为可见的动作或表情
- "气氛变得紧张" → 改为具体的环境变化或角色的身体反应

**以下内容属于正常的视听语言，不需要修改：**
- 特效描述（"画面扭曲"、"时间倒流"）——这是视觉呈现
- 内心独白/旁白（`角色（OS）：`）——这是声音呈现
- 音效/配乐指示——这是听觉呈现
- 字幕/文字画面——这是视觉呈现

视听化修正是小手术，不改变任何叙事内容，只是把"不可拍"翻译为"可拍"。

### 四个微调焦点

视听化修正之后，进入导演层面的剧本微调。微调集中在以下四个焦点，每个焦点都服务于已确认的导演定调和节奏规划：

**焦点一：表演节奏**

看每场戏中角色的动作节奏是否与目标节奏匹配。核心手段是拆分或合并动作：

- **拆分动作以延长节奏**：把一个笼统的动作拆成多个具体的小动作，增加时间容量和表演层次。
  - 原文："她打开信，看完后放下。"
  - 微调："她拿起信封，翻过来看了看背面。撕开封口，抽出信纸。展开。视线从上往下移动。她把信折好，放回信封，放在桌上。手指在信封边缘停了一下。"
  - 理由：这是一个情感重场，需要通过动作的细分让观众有时间感受角色的情绪变化。

- **合并动作以缩短节奏**：把多个碎步动作压缩为一个干脆的动作，加快段落速度。
  - 原文："他站起来，走到门口，打开门，走出去，关上门。"
  - 微调："他起身出门。"
  - 理由：这里不是表演重点，角色离开的动作不需要分解。

**焦点二：台词对白**

看台词是否与目标节奏和角色状态匹配。核心手段：

- **延长台词**：在关键情感节点增加台词的犹豫、重复、岔开，让角色"说不到点上"的过程本身成为表演。
- **缩短台词**：把冗长的解释压缩为一个词或一句话，甚至删掉台词用沉默替代。
- **改写内容**：在不改变信息传递的前提下，调整台词的表达方式——更口语、更符合角色此刻的情绪状态。
- **调整断句**：同一句话的断句位置影响节奏——"我不想……走"和"我——不想走"的节奏完全不同。断句也影响画面剪辑点——每个断句都是一个可能的切镜时机。

**焦点三：画面内容**

看画面中是否有足够的视觉细节支撑叙事和情感，或者是否有多余的画面拖慢节奏。核心手段：

- **增加细节动作**：在情感重场中增加角色的微小动作和环境细节，提升画面质感。如《甜蜜蜜》中张曼玉听到消息后不是直接哭，而是先笑了一下再崩溃——这个"先笑后哭"就是导演增加的细节表演，不在剧本的"常规反应"中，但它让角色从纸面立了起来。
- **删除多余画面**：如果一场戏中有某个画面既不推进情节也不传递情感也不建立氛围，删掉它。
- **替换画面**：同一个叙事目的可以用不同的画面实现——选择视觉冲击力更强或更契合导演定调的那个。

**焦点四：意象落地**

意象体系（视觉母题、环境意象、突出强化的视觉元素）已在第一步导演定调中确定。本焦点的任务是把已确定的意象**具体植入剧本**：

- 在哪些场景中植入视觉母题的第一次、第二次、第N次出现？
- 哪些场景需要增加与角色状态呼应的环境细节？
- 需要突出强化的视觉元素是否在剧本中有足够的呈现空间？

这不是"重新设计意象"——而是把已规划好的意象在剧本文本层面落实到位。

### 微调不做的事

- **不改变故事的核心戏剧动作**（那是编剧阶段的事）
- **不改变人物弧光的方向**（除非发现结构性问题）
- **不添加新的主要角色或主要情节线**
- **不把正常的视听语言（特效、旁白、字幕等）当作"非视听"来强行修改**

### 微调的输出方式

微调按四个焦点分组输出，每处微调用对比格式：


## 五、工作流纪律

### 步骤间暂停

每一步完成后暂停，等待用户的指令：
- **[通过]** → 进入下一步
- **[修改]** + 具体意见 → 修改后重新输出当前步骤
- **[自检]** → 按当前步骤的检查清单逐条评估并输出报告

### 步骤间不跳步

不允许在导演定调未确认的情况下直接开始分镜拆解。不允许在节奏规划未确认的情况下直接开始生成分镜表。

### 输出前自检

所有内容在输出给用户之前，先内部自检一遍（不展示自检过程）。自检依据：
- 当前步骤的检查清单
- 写作红线（继承自编剧技能：不写心理描写、括号暗示、解释性台词等）
- 叙事目的的完整性
- 节奏标注的合理性
