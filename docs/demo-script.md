# Novel-to-Script 演示脚本

总时长：约 2 分钟

---

## 第 1 段 — 开场 + 主题切换（0:00–0:15）

| 镜头 | 操作 |
|------|------|
| 全屏展示首页 | 浏览器打开 `http://localhost:5173` |
| 点右上角 🌙 按钮 | 切到深色主题 |
| 再点 ☀️ 切回来 | 恢复浅色 |

**念稿**：

> "Novel-to-Script 是一款 AI 辅助小说转剧本工具，支持浅色和深色双主题。左边是上传与设置区，中间是剧本预览，右边是分集设置。"

---

## 第 2 段 — 加载示例 + 转换（0:15–0:45）

| 镜头 | 操作 |
|------|------|
| 点 **🎲 加载示例小说** | 文件列表出现 1 个文件 |
| 标题输入框展示 | "穿成病娇大佬的亡妻" 已自动填入 |
| 右边目标集数填 **3** | 滚动到右侧分集设置区 |
| 点 **开始转换** | 等待进度条走完（约 2 秒）|

**念稿**：

> "点击加载示例小说，三章穿书题材文本自动填充。选择网文漫剧模式，设置目标三集，点击开始转换。页面实时显示转换进度——场景检测、构建场景、规划集数，每个阶段耗时一目了然。"

---

## 第 3 段 — 预览分镜（0:45–1:05）

| 镜头 | 操作 |
|------|------|
| 中间区域切到 **🎬 分镜预览** tab | 点击第二个 tab |
| 点集号切换 | 依次点"第1集""第2集""第3集"|

**念稿**：

> "转换完成后可以在剧本预览和分镜预览之间切换。分镜视图支持按集筛选，每张卡片展示场景地点、时间和情绪弧线。"

---

## 第 4 段 — YAML 源码 + 下载（1:05–1:25）

| 镜头 | 操作 |
|------|------|
| 切到 **📝 YAML 源码** tab | 展示语法高亮的 YAML |
| 滚动展示结构 | 看到 characters / scenes / episodes 等区段 |
| 点 **下载 YAML** | — |

**念稿**：

> "YAML 源码带有语法高亮，结构清晰——角色表、场景表、节拍序列按层级组织。输出的 YAML 干净无冗余字段，可直接用于后期制作流程。点击下载即可导出。"

---

## 第 5 段 — 封面生成（1:25–1:45）

| 镜头 | 操作 |
|------|------|
| 看右侧某集卡片 | 滚动到分集设置下方的结果区 |
| 点 **🎨 生成封面** | 封面提示词出现 |
| 点 **📋 复制提示词** | — |

**念稿**：

> "每集还可以一键生成封面提示词，根据剧情钩子和角色信息自动组合中英文 AI 绘图提示词。点击复制即可粘贴到 Midjourney 或 DALL·E 中使用。"

---

## 第 6 段 — 收尾（1:45–2:00）

| 镜头 | 操作 |
|------|------|
| 切到 GitHub 仓库 README | 浏览器新标签页 |
| 展示 Vercel 在线链接 | — |
| 滚动展示测试/CI/Docker | 快速滑过 |

**念稿**：

> "项目已开源在 GitHub，包含 70 个自动化测试、GitHub Actions 持续集成、Docker 一键部署。同时部署在 Vercel 上，打开浏览器即可在线体验。感谢观看。"
